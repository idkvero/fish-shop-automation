# Автоматизация учета в магазине аквариумных рыбок

Проект представляет собой конфигурацию на платформе **1С:Предприятие 8.3** для оперативного учета и расчета себестоимости.

## Что реализовано в проекте:
* **Архитектура данных:** Настроена структура справочников (*Аквариумы*, *Породы рыб*) и документов (*Покупка рыб*, *Покупка корма*, *Продажа рыб*).
* **Оперативный учет:** Создан регистр накопления *«Остатки рыб»* для точного контроля движений в реальном времени.
* **Проведение документов:** * Реализован алгоритм для документа *«Покупка корма»* с динамическим распределением стоимости корма по себестоимости рыб, находящихся в конкретном аквариуме.
  * Настроен контроль остатков при *«Продаже рыб»* (защита от ухода в минус) с расчетом средней себестоимости списываемых партий.
* **Отчетность:** Разработан аналитический отчет *«Прибыль от продаж»* на базе **СКД**, выводящий финансовый результат (Выручка, Себестоимость, Чистая прибыль) с профессиональным форматированием данных.

## Инструкция по развертыванию:
Для проверки работы конфигурации необходимо загрузить стандартный файл информационной базы **.dt** через Конфигуратор (*Администрирование -> Загрузить информационную базу*).
